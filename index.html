This project is an experimental project for learning WebSocket and WebRTC.

WebSockets_WebRTC

  • This project is developed by Rustu Seyhun Sariyildiz.
  • It's a video chat application implemented with chat rooms.
  • It's only available for Chrome/Chromium with WebRTC support.
  • 'node.js' is used with 'socket.io'
  • The latest update of the project is deployed at http://www.html5cam.com

Requirements

  • node.js
  • a HTTP server
  • Chrome/Chromium with WebRTC support
